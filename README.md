# Learning Python: 

Welcome to my Python learning journey! 🐍  
This repository contains my notes, code examples, and exercises as I learn Python from scratch.  
Currently, I have covered **variables** and **typecasting** – the foundational building blocks of Python programming.

---

## 📚 Topics Covered So Far

### 1. Variables
- What are variables and how to assign values
- Naming conventions (snake_case, meaningful names)
- Dynamic typing – assigning different data types to variables
- Multiple assignment and swapping values

### 2. Data Types
- `int` – integers
- `float` – floating point numbers
- `str` – strings
- `bool` – boolean values
- `NoneType` – representing absence of value

### 3. Typecasting (Type Conversion)
- Implicit vs explicit conversion
- Using built-in functions:
  - `int()`
  - `float()`
  - `str()`
  - `bool()`
- Common pitfalls (e.g., converting non‑numeric strings to numbers)
